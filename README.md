
# Archart - Responsive Modern Web App

**Archart** is a responsive website designed for showcasing modern architectural designs and projects. This project highlights clean, professional layouts using only **HTML** and **CSS** to create a sleek and user-friendly interface that adapts seamlessly across all devices.

## Features

- **Fully Responsive:** Designed to look great on all screen sizes, from smartphones to large desktops.
- **Modern Design:** Incorporates contemporary web design principles with a focus on simplicity and usability.
- **Cross-Browser Compatibility:** Ensures consistent performance across different web browsers.

## Live Demo

Explore the live version of the Archart website: [Archart Live Demo](https://onkar49.github.io/archart-resonsive/)

## Getting Started

To clone and run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/onkar49/archart-resonsive.git
   ```
2. Navigate to the project directory:
   ```bash
   cd archart-resonsive
   ```
3. Open `index.html` in your preferred web browser to view the website.

## Technologies Used

- **HTML5**
- **CSS3**
